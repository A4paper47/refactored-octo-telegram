Tips & Workflow

Translator (DM)

  • Send .srt in DM (no caption needed).
  • Best filename: MOVIECODE.srt (example: BN-260303-01.srt) for perfect matching.
  • After submit: system creates Queue row + (optional) forwards to outbox.

VO (Group)

  • Upload media/zip and put movie code + role + lines in caption.
  • Example: BN-260303-01 man-1 120
  • ZIP supported: filenames containing man1/man-1/fem2/fem-2 etc are auto-detected.

Deadlines

  • Late/Overdue only counts if deadline exists.
  • If deadline_at is empty/NULL → NOT late.
  • Use workload pages to set deadlines and send reminders.

Backups & Restore

  • Go to Backups to download: JSON ZIP (best for restore), Excel (sharing), logs.txt (debug).
  • To send backups to Telegram: run /backup_here in the target chat, then /backup_now dest json.
  • To restore: go to Restore → upload JSON ZIP → dry run → choose Replace or Append → type RESTORE.
